# Decision Trees and Random Forest - Lab 9

## Objective

The objective of this lab is to understand how Decision Tree and Random Forest algorithms work and how to use them for classification tasks. The lab also focuses on evaluating model performance using different metrics.

## Dataset

The dataset used in this lab contains information about loans. The goal is to predict whether a loan will be fully paid or not based on the given features.

## Methods Used

* Data preprocessing and splitting the dataset into training and testing sets.
* Building a Decision Tree classifier.
* Building a Random Forest classifier.
* Evaluating model performance using:

  * Confusion Matrix
  * Classification Report
  * Accuracy Score

## Results

The Random Forest model generally performed better than the Decision Tree model.
The model showed strong performance in predicting class 0 but weaker performance in predicting class 1 due to class imbalance in the dataset.

## Conclusion

Decision Trees and Random Forest are powerful classification algorithms. Random Forest usually provides better accuracy and more stable results compared to a single Decision Tree. However, model performance can be affected by imbalanced data.

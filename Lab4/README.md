# Lab 4 - Data Quality Assessment & Preprocessing

This repository contains my solution for Lab 4.

In this lab, I applied data preprocessing techniques on the Netflix dataset, including:

- Identifying data quality issues  
- Handling missing values using the Remove Records strategy  
- Detecting and removing outliers using the IQR method  
- Applying normalization using Min-Max and Z-score  
- Evaluating correlation between features and deciding whether to apply PCA  

The results show that PCA was not applied due to weak correlation between numerical features.

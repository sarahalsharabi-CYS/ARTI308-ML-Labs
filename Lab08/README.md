# K-Nearest Neighbors (KNN) Classification Project

## Description
This project applies the K-Nearest Neighbors (KNN) algorithm to classify data into two classes. 
The dataset was first standardized using StandardScaler, then split into training and testing sets.

## Steps Performed
1. Loaded and explored the dataset.
2. Standardized the features using StandardScaler.
3. Split the data into training and testing sets.
4. Trained a KNN model with different K values.
5. Plotted the error rate versus K values.
6. Selected the best K value based on the lowest error rate.
7. Retrained the model using the selected K value.
8. Evaluated the model using confusion matrix and classification report.

## Best K Value
The chosen K value was:

31

## Final Result
The model achieved an accuracy of approximately:

0.84 (84%)

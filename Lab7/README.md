Lab 7 - Logistic Regression

In this lab, we applied Logistic Regression to predict whether a user clicked on an advertisement based on several features such as age, income, and internet usage.

The dataset used is advertising.csv.

Steps performed in this lab:
- Data loading and exploration
- Data visualization using seaborn
- Splitting the data into training and testing sets
- Training a Logistic Regression model
- Making predictions
- Evaluating the model using classification report

Files included:
- Lab7.ipynb
- advertising.csv

The model achieved an accuracy of approximately 91%.

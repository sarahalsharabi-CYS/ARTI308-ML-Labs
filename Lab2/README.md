# Lab 2: Student Performance Dataset

**Dataset:** Student Performance Dataset  
**Source:** Kaggle https://www.kaggle.com/datasets/devansodariya/student-performance-data  

**Problem Type:** Regression  
**Target Variable:** G3  

## Problem Statement
The goal of this project is to predict the final grade of students based on demographic, social, and academic features.

## Description
This dataset contains 395 students with 33 features including demographic, social, and academic information such as parents' education, family relations, alcohol consumption, marital status of parents, and family size.  
The dataset can be used for regression and data analysis tasks.

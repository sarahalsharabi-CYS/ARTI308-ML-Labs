# Lab10 - Support Vector Machines (SVM)

This project is a machine learning lab that applies **Support Vector Machine (SVM)** classification on the famous **Iris dataset**.

## Project Overview

The notebook analyzes iris flower measurements and builds an SVM model to classify the flower species into:

- Iris-setosa
- Iris-versicolor
- Iris-virginica

## What the Notebook Includes

- Loading and exploring the Iris dataset
- Data visualization using Seaborn and Matplotlib
- Pairplot and KDE plot for exploratory data analysis
- Splitting the data into training and testing sets
- Training an SVM classifier using `SVC`
- Evaluating the model using:
  - Confusion Matrix
  - Classification Report
- Improving the model using `GridSearchCV`

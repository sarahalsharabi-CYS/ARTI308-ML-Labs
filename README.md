# Student Performance Dataset – Lab 2

**Dataset:** Student Performance Dataset  
**Source:** UCI Machine Learning Repository  
**Problem Type:** Regression  
**Target Variable:** G3  

**Description:**  
The dataset contains 395 students with 33 features including demographic, social, and academic information.  
The goal of this project is to predict the final grade (G3) of students based on their demographic and academic attributes.
